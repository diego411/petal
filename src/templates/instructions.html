Overview

This system is used to collect electrical signal data for plants. The signals are collected through a sensor. Currently only the Oxocard sensor is supported. For instructions to set up the Oxocard please refer to the Oxocard Setup page.

Recordings
If you are trying to collect data for your own experiments, you can create recordings. Instructions for this can be found on the Recordings page

Experiments
If you want to participate in our experiment to collect plant signal data for emotion recognition please refer to the Experiments page for instructions.

Scripts
If you are searching for instruction on the scripts which can be run on the sensors please refer to the Scripts page.

Error codes
If you are encountering error codes and don't know what they mean please refer to the Error codes page.
