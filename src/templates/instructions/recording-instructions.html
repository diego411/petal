Recordings


Before being able to work with recordings please set up your Oxocard sensor according to the instructions here

Create Recording

You can create a recording by restarting your Oxocard and then pressing start

When you then navigate to the Recordings page you should see a unit for your recording.

Note: if there already is an existing recording for your Oxocard restarting the oxocard will not create a new one. If you want a fresh recording you will first have to get rid of the current recording for that Oxocard. For more information on how to get rid of recordings refer to the Finish Recording section

Start Recording

In the top right of the unit there is a button to start the recording.

The unit should now display a start date and an estimate on when the next update should arrive.

Once measurement data comes in it will be display inside the graph of the unit


Finish Recording

To finish a recording you simply press the red button in the top right of the unit. Once you confirm that you want to finish and the loading is done the data will be uploaded to the dropbox. To access the data simply click the link that will be displayed on the bottom of the screen.
